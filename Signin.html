-Student Housing Management System

The Student Housing Management System is designed to simplify the housing search for students at the University of Idaho.
The system features a responsive web interface, robust backend logic, and API integration to verify student details, supported by a comprehensive database for housing preferences and listings.
It comprises user management for registration and profiles, property listings, student preference settings, a bias-free matching algorithm, payment processing, relocation services, and communication tools to connect students with agents.
Developed on a web platform using HTML, CSS, Bootstrap 5, and JavaScript for the front end, with Flask for the backend, and MySQL for database management. Local development was facilitated by XAMPP, and payments are processed through a secure integration.

WeChat Mini Programs on Campus Second-hand Platform

Aimed to enhance the efficiency and convenience of second-hand goods trading on campus, addressing existing challenges through a comprehensive solution.
Developed a Wechat Mini Platform that supports user registration, product listings, browsing, communication, and e-payment, facilitating a seamless trading experience.
The project was executed with a keen focus on data analysis for understanding user behavior and pricing trends, alongside the meticulous design of UI/UX to optimize user interaction and satisfaction.
Utilized a blend of data mining methods, machine learning algorithms for backend analysis, and robust front-end development tools to bring the platform to life, ensuring a responsive and intuitive user experience.
